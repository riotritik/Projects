About This Project

This Email Spam Detection project is built using Django for the backend, and integrates advanced machine learning libraries such as TensorFlow (v2.18.0), Scikit-Learn, Pandas, and NumPy.

The system uses a hybrid detection approach combining Naive Bayes and Bag of Words models to effectively identify and classify spam messages. It was developed through a blend of AI-assisted coding and my own understanding of data science and NLP concepts.

The purpose of this project is to showcase practical implementation of spam filtering using both traditional ML and intelligent rule-based techniques.
